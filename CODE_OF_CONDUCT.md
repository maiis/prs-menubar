# Code of Conduct

This project adheres to the Contributor Covenant.

## Pledge
We strive to foster an open, welcoming, harassment-free community.

## Standards
Be respectful, assume good intent, provide constructive feedback.

## Unacceptable Behavior
Harassment, personal attacks, discriminatory language, or publishing others' private information.

## Enforcement
Report issues via project issues or by contacting the maintainer.

Repeated or severe violations may result in temporary or permanent exclusion.
