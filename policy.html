Privacy Policy for PRs MenuBar

Effective Date: 2025-01-28

PRs MenuBar ("we", "our", or "app") respects your privacy. This Privacy Policy explains how we collect, use, and protect data when you use the PRs MenuBar app.

1. Information We Collect

PRs MenuBar integrates with GitHub's API. With your GitHub Personal Access Token, we access:

  • Pull requests awaiting your review
  • Basic PR information (title, repository, author, creation date)

We do not collect any data beyond what is necessary to display your pending pull requests.

2. How We Use Your Data

  • To fetch and display pull requests that require your review
  • To provide a convenient menu bar interface for GitHub notifications
  • All data remains only on your device
  • Your GitHub token is stored securely in the macOS Keychain

3. Sharing Your Data

  • We do not sell, trade, or share your GitHub data with third parties
  • We do not send your GitHub token or any data to our servers
  • All API requests are made directly from your device to GitHub's servers

4. Security

Your GitHub Personal Access Token is stored securely in the macOS Keychain, which provides industry-standard encryption. We never transmit your token to any server other than GitHub's official API endpoints.

5. Your Choices

  • You can delete your token at any time via the app's Settings
  • You can revoke the token at GitHub Settings
  • You can delete all app data by uninstalling the app

6. Contact

For questions about this Privacy Policy, please contact us at: apps@maiis.me
